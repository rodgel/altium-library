The script contained in this zip file can be used to prevent an invalid DRC error - it will allow short-circuits in vias and pads in the board cut-out area of Rosenberger edge-mount connectors. When run, it will append these short-circuit exceptions to your design rules.

For more detailed information please see the related techdocs article:
http://techdocs.altium.com/display/SCRT/Design+Rule+for+Edge-Mount+Connectors
